# Project Astra – Rocket Engineering Website

This repository contains the frontend website for Project Astra, a student-led rocket engineering initiative by IEEE MUJ.

The website showcases the overall project including rocket design, simulation, mathematical modeling, and team structure.

---

## Project Overview

Project Astra focuses on building and analyzing a model rocket system by combining:

- Rocket structure and design
- Propulsion and aerodynamics
- MATLAB-based flight simulation
- Mathematical modeling
- Team collaboration

This website acts as a central platform to present the project's progress and technical aspects.

---

## Tech Stack

- Frontend: React + Vite
- Styling: Tailwind CSS
- Graphs (planned): MATLAB outputs
- Future Integration: Simulation API
